## 基本语法

```java
if (condition) {
  // 满足时执行
} else if (condition) {
  // 次条件
} else {
  // 否则
}
```

```java
public class Main {
    public static void main(String[] args) {
        int n = 70;
        if (n >= 60) {
            System.out.println("及格了");
        }
        System.out.println("END");
    }
}
```
